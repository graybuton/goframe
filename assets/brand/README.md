# GoFrame Brand Assets

This directory contains lightweight exported GoFrame logo assets for README,
documentation, repository UI, and maintainer use.

The GoFrame mark is a custom geometric `G` with a snake-eye cut and a
tail-like detail. The mark is intended to suggest flexibility, motion, and a
Go-first framework/toolchain identity.

The geometry is Jost-inspired and manually modified for GoFrame. These assets
are project branding files, not a claim of affiliation with any third-party
brand.

GoFrame is not affiliated with Google LLC or the Go project.

## Files

- `goframe-mark.svg` - scalable mark/icon.
- `goframe-mark-128.png` - 128px raster mark export.

## Usage

Prefer SVG in README and docs. Use PNG only where SVG is unsupported. Keep the
mark with enough padding and do not recolor it in a way that implies
affiliation with another project.
